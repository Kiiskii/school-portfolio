Muuta fonttikokoa ja tyyliä Javascriptillä

Huom! Kaikki muutokset tapahtuvat Javascriptin avulla

Listan fonttikokoa ja tyyliä muutetaan alla olevia nappeja painamalla. Listan oletus fonttikoko on 10pxja oletustyyli on "Courier New".

Tämän listan fonttikoko muuttuu napista painamalla

Fonttikoko nyt 16px

Fontti on nyt: "Courier New"

  1. Ensimmäinen napinpainallus muuttaa fonttikooksi 12px
  2. Toinen napinpainallus muuttaa fonttikooksi 14px
  3. Kolmas napinpainallus muuttaa fonttikooksi 16px
  4. Neljäs napinpainallus muuttaa fonttikooksi 18px
  5. Viides napinpainallus muuttaa fonttikooksi 10px
  1. Ensimmäinen napinpainallus muuttaa fonttityypiksi "Roboto"
  2. Toinen napinpainallus muuttaa fonttityypiksi "Manslava"
  3. Kolmas napinpainallus muuttaa fonttityypiksi "Grenze"
  4. Neljäs napinpainallus muuttaa fonttityypiksi "Turret road"
  5. Viides napinpainallus muuttaa fonttityyiksi "Courier New"
